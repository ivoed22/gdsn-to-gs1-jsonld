# v0.7.1 — Streamlit Cloud Import Fix

## Overview

Version v0.7.1 is a patch release that fixes Streamlit Cloud startup
robustness after v0.7.0.

The issue was related to UI imports during Streamlit Cloud startup. The app now
uses package-qualified UI imports and regression tests verify that the
Streamlit app can import UI helpers from a non-repository working directory.

## Fixed

* Made `app/streamlit_app.py` import-safe with package-qualified `app.ui`
  imports.
* Added a `main()` guard for safer Streamlit startup.
* Added `app/__init__.py`.
* Added regression tests for Streamlit UI imports.
* Fixed CI-only import path handling in the regression tests.

* Mapping YAML files are unchanged.
* Catalog data is unchanged.
* No warnings were suppressed.
* No semantic mappings were changed.
* No new dependencies were added.
