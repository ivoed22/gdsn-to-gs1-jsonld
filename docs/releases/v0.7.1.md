# v0.7.1 — GitHub Standards Review Issues

## Overview

Version 0.7.1 connects the six open standards decision records to GitHub
Issues. This makes each standards and governance question easier to assign,
discuss, reference, and track while the Markdown SDR remains the maintained
decision record.

## Added

- GitHub Issues for SDR-001 through SDR-006.
- Direct issue links in each SDR and the standards decision register.
- Issue numbers and URLs in the JSON and CSV standards-review backlogs.
- Release and regression documentation for the linked review workflow.

## Issue tracking

- SDR-001: nutrient modelling.
- SDR-002: document and DPP modelling.
- SDR-003: image representation.
- SDR-004: allergen containment.
- SDR-005: certification semantics.
- SDR-006: YAML and catalog governance.

Existing issues were reused where they already represented the same open
standards question. New issues were created for the remaining decisions.

## Preserved

- Converter output is unchanged.
- Mapping YAML files are unchanged.
- Catalog data is unchanged.
- No unresolved warning is suppressed.
- All standards decisions remain open until an authorized review records a
  decision.
