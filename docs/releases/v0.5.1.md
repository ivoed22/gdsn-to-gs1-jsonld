# v0.5.1 — Streamlit UI Polish

## Overview

Version v0.5.1 is a patch release focused on Streamlit UI polish, stakeholder
demo readiness and documentation improvements.

This release does not change converter logic, mapping logic, validation
behavior, CLI behavior or generated output. It improves how the existing
converter is presented and reviewed in the Streamlit app.

## Added

Added documentation explaining the strategic value of the project for GS1

- internal positioning

- strategic next steps
- stakeholder one-pager
- demo story
- before/after example
- AI relevance

Improved the post-conversion review experience with:
