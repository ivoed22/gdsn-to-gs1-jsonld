# v0.7.0 — Standards Review Backlog

## Overview

Version 0.7.0 converts the 12 remaining non-blocking conformance warnings into
six structured standards and governance review topics.

## Added

- Standards decision register under `docs/standards-decisions/`.
- Six open SDRs for nutrient, document/DPP, image, allergen, certification,
  and YAML/catalog governance decisions.
- JSON and CSV machine-readable backlogs.
- Offline `export-standards-backlog` command.
- Recommended owners, target releases, options, impact analysis, and follow-up
  actions for each decision.
- Compact Streamlit standards-review status.

## Governance

All decision records remain Open. The release does not treat a recommendation
as an accepted standards decision. Future status values are Proposed,
Accepted, Rejected, or Deferred and require explicit review.

## Preserved

- Converter output is unchanged.
- Semantic mappings and mapping YAML files are unchanged.
- Catalog data is unchanged.
- No unresolved warning is suppressed.
- Existing warnings remain non-blocking and standards-review-required.
- No new dependencies were added.
