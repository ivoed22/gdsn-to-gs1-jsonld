# v0.9.1 — Public Source Data Inventory & Reference Import

## Overview

Version v0.9.1 adds a public source data inventory and offline reference-data
import command. It prepares authoritative source references for future manual
JSON-LD prototyping and mapping-candidate review without implementing those
future features yet.

## Added

- Added `reference_data/source_manifest.json` with public source URLs,
  retrieval metadata, local paths, checksums, usage notes, and rights notes.
- Added `reference_data/schemas/source_manifest.schema.json`.
- Added a public GDSN BMS/XPath workbook copy under `reference_data/raw_public/`.
- Added normalized reference outputs under `reference_data/normalized/`.
- Added `src/gdsn_to_gs1_jsonld/reference_data_importer.py`.
- Added `gdsn-to-gs1-jsonld import-reference-data`.
- Added regression tests for WebVoc UTF-8 BOM loading, WebVoc class/property
  extraction, fake GDSN Excel normalization, candidate/deleted row flags,
  checksums, summary JSON, CLI outputs, and source-manifest schema coverage.
- Added documentation in `docs/source-data-inventory.md` and
  `docs/reference-data-import.md`.
