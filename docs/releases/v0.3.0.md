# v0.3.0 — BMS/XPath-aligned Certification & Document Mapping

Version 0.3.0 adds experimental certification and referenced-document output
designed from the GDSN 3.1.36 BMS/XPath mapping catalog and locally validated
GS1 Web Vocabulary terms.

## Added

- `mapping/mapping_v0_3.yaml`
- typed certification and referenced-document canonical models
- `gs1:certification` output with `gs1:CertificationDetails`
- certification standard, value, audit date, start date, and end date
- experimental identifier, issuance date, and agency-identifier alignments
- DPP-like and certification document objects
- mapping catalog governance, catalog documentation, and design documentation
- Certifications & Documents v0.3.0 Streamlit profile
- v0.3.0 CLI, Streamlit, compatibility, catalog, and unmapped-report tests

## Preserved

- v0.1.0 JSON-LD output with `mapping_mvp.yaml`
- v0.2.0 JSON-LD output with `mapping_v0_2.yaml`
- shared package logic for CLI and Streamlit

## Limitations

Generic document relationships remain experimental because the validated GS1
Web Vocabulary has no direct generic referenced-document property. No
certificate verification, URL dereferencing, resolver call, DCAT/DPROD model,
Verifiable Credentials (VC), or full GDSN XSD validation is included.
